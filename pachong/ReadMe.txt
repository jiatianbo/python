一、运行环境：

Ubuntu 16.04
pyCharm 社区版
python 2.7.12

二、功能介绍

爬取网站http://www.btba.com.cn/上的种子文件

三、使用说明：

1、在Ubuntu下输入“python bt.py”即可运行

2、爬取的种子将存放在pachong文件夹中

说明：1、由于爬取的种子文件所以没有采用数据库
      2、由于时间关系只爬取了1000多个种子，作为演示

四、其他：

1、发现的问题：爬取过程中出现爬取中断
   解决方案：路由器导致无线网络不畅通，重启路由器，确保网络畅通
2、注意python 2和python3的某些语法细节上的区别
